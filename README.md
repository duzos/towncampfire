## What is it?

A Forge mod that turns campfires into the heart of a village. Place a **town campfire** to mark out a settlement, give it a name, and the mod keeps track of the villagers who belong to it.

## Features

- **Coloured town campfires** — blue, green, red, white, and yellow variants.
- **Found a town** — a town campfire defines and names a village.
- **Villager tracking** — the mod knows which villagers belong to each town.
- **Server-saved** — village data persists with the world.

## Requirements

- Minecraft **1.19.2**
- Minecraft **Forge**
